# About
This is a school project.
It's a matchmaking server where user can connect to and get transfared to a lobby with up to 20 Players.
The users get ranked from:
- Total hours
- Kills in the last game
- Last win
- Last Position

There are two projects:
- matchma-client: a simple testing client
- matchma-server: the server. For deployment, follow the instructions inside [here](./matchma-server/README.md)
